# Sales Forecasting & Demand Intelligence

## Project Overview

This project analyzes historical retail sales data and provides business insights through forecasting, anomaly detection, and product segmentation. An interactive Streamlit dashboard allows users to explore sales trends and make data-driven decisions.

## Features

- Sales Dashboard
- Sales Forecasting
- Anomaly Detection
- Product Segmentation
- Interactive Visualizations
- Business Recommendations

## Technologies Used

- Python
- Pandas
- NumPy
- Streamlit
- Plotly
- Scikit-learn
- XGBoost
- Statsmodels
- Prophet
- Matplotlib

## Dataset

- Superstore Sales Dataset
- Video Game Sales Dataset (for product segmentation)

## How to Run

Install the required libraries:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
